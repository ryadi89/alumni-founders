About these graphs

The data source

These graphs represent data that was originally referenced in an October 2013 PitchBook blog post called Top Universities Producing VC-backed Entrepreneurs.

Start-up company description, founding date, and VC-exit details were primarily collected from self-reported data in Crunchbase.

UC Berkeley alumni founder degree information was primarily collected from self-reported data on LinkedIn.

The csv file behind these graphs is here

The technology

These interactive graphs were made using an excellent javascript charting library called dc.js, which was built to work natively with crossfilter and leverages the super awesome d3.js.
